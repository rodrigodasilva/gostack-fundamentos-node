## üèÅ Run this project:

To run on the first time,
into your folder:

```bash
$ git clone https://github.com/rodrigodasilva/gostack-fundamentos-node
```

Into repo folder, to intall `Node_Modules` run:

```bash
yarn
```

And start the server:

```bash
yarn dev:server
```

To run the tests do:

```bash
yarn test
```
